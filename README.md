# Number Guessing Game (Python)

A fun and interactive terminal-based game where you have to guess the secret number chosen by the computer.

## Features
- **Dynamic Difficulty:** Choose between 'Easy' and 'Hard' modes.
- **Input Validation:** Prevents the game from crashing if you enter non-numeric characters.
- **Limited Attempts:** You have 5 chances to guess correctly!

## How to Run
1. Make sure you have **Python** installed on your computer.
2. Run the following command:
   ```bash
   python game.py
